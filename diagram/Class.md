## Глоссарий

| Понятие           | Описание                                                                 |
|-------------------|--------------------------------------------------------------------------|
| **Identifiable**  | Интерфейс, определяющий метод получения идентификатора сущности         |
| **BaseEntity**    | Абстрактный базовый класс для всех сущностей системы с общими атрибутами и методами |
| **User**          | Класс пользователя системы: атрибуты - username, email, passwordHash; методы - создание проектов, валидация данных |
| **Project**       | Класс проекта: атрибуты - name, description, status; методы - расчет прогресса, валидация данных |
| **Task**          | Класс задачи: атрибуты - title, description, priority, status; методы - проверка просрочки, валидация данных |
| **ProjectStatus** | Перечисление статусов проекта                                            |
| **TaskPriority**  | Перечисление приоритетов задачи                                          |
| **TaskStatus**    | Перечисление статусов выполнения задачи                                  |
